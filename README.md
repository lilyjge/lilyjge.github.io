# Portfolio 
This is my portfolio website made with pure HTML, CSS, and Javascript.

See it deployed [here](https://lilyjge.github.io/)!

## Local preview

Run a tiny local server from the repo root:

```powershell
python -m http.server 8000 --bind 127.0.0.1
```

Then open <http://127.0.0.1:8000/>. This is better than opening `index.html` directly because local links, nested pages, and API requests behave more like the deployed site.

P.S. Fun fact: my favicon is an SVG generated by Gemini 3 Flash Preview Thinking, manually edited by me with this [editor](https://www.svgviewer.dev/), and converted into proper favicon format with this [tool](https://svg2ico.com/convert).
